# CSL стиль цитирования для журнала «Loci Imperiales» (ГОСТ Р 7.0.5-2008)

Стиль цитирования предназначен для использования в библиографических менеджерах типа **Zotero** и **Mendeley**. Он записан с помощью Citation Style Language (CSL). Основан на ГОСТ Р 7.0.5-2008 «Библиографическая ссылка. Общие требования и правила составления».

Стиль разработан для публикаций в научном журнале [**«Loci Imperiales»**](https://peterhofmuseum.ru/spec/li), издаваемом в ГМЗ «Петергоф». Позволяет корректно цитировать литературу и архивные источники в подстрочных сносках (Notes), а также составлять список использованной литературы (Bibliography).

        - Либо откройте Zotero > Правка > Настройки > Цитирование > Добавить из файла.

## Сортировка в списке литературы

Сортировка в библиографии производится по следующей иерархии:

1.  **Язык:** Сначала издания на русском языке, затем — на иностранных.
2.  **Автор / Заглавие:** Второй уровень сортировки.
3.  **Дата издания:** Третий уровень сортировки.

## Локализация

Реализована локализация стандартных обозначений на русский язык со следующими особенностями:

- **Составитель:** Выводится при заполненном поле «Редактор серии» (`collection-editor`).
- **Публикатор:** Выводится при заполненном поле «Контрибьютор» (`contributor`).

> **Примечание:** Локализация английских терминов на данный момент не реализована.

## Особенности форматирования

- **Курсив:** Имена авторов выводятся курсивом (`italic`).
- **Неразрывные пробелы (`&#160;`):** Применяются в следующих случаях:
    - Между инициалами и между фамилией и инициалами авторов.
    - Перед разделителями: косой чертой (`/`) и двойной косой чертой (`//`).
    - В численных данных: `Т.&#160;`, `№&#160;`, `С.&#160;`, `&#160;с.`
    - В повторных ссылках: `Там&#160;же.`
- **Комплексные ссылки:** Разделяются точкой с запятой (`;`). Группировка ссылок на одного автора в настоящее время не поддерживается.

## Положение ссылки (Цитирование в сносках)

Стиль различает положение ссылки:

| Тип ссылки | CSL Параметр | Поведение |
| :--- | :--- | :--- |
| **Первичная** | `first` | Выводит полные данные, **кроме** поля «Издатель» (`publisher`) для книг, глав и статей в сборниках конференций. |
| **Повторная** | `subsequent` | Выводит поле «Краткое назв.», если оно заполнено. В противном случае выводится полное название. |
| **Там же** | `ibid` | Повторная ссылка, идущая **сразу** после предыдущей (без других ссылок между ними). Выводит `Там же`. |
| **Там же с локатором** | `ibid-with-locator` | Аналогично `ibid`, но с указанием другой страницы (диапазона). |

> **Техническое ограничение:** Нельзя настроить градацию для повторных ссылок. Если в новой ссылке отличается хотя бы один элемент (кроме страниц), она не будет считаться `ibid` и обработается как `first` или `subsequent`.

## Типы записей и особенности их обработки

Стиль учитывает следующие типы элементов в Zotero:

### 1. Книга (`book`)
- Учитываются поля: Автор, Редактор, Составитель, Переводчик, Публикатор.
- **Многотомные издания:** Указывается общее количество томов и данные конкретного тома (при наличии).
- **Сериальные издания:** Указывается название серии и номер выпуска.

### 2. Глава книги (`chapter`) и Статья в сборнике конференции (`paper-conference`)
- Если при цитировании не указан локатор (страница), автоматически выводится диапазон страниц из поля «Страницы» (Pages) самой главы/статьи.

### 3. Статья в журнале (`article`)
- Учитываются: Автор, Название статьи, Название журнала, Том, Номер, Год издания.
- Если локатор не указан, выводится полный диапазон страниц статьи.

### 4. Диссертация и автореферат (`thesis`)
- **Обозначение:** Заполняется в поле **«Тип»** (`genre`). Пример: `автореф. дис. ... канд. ист. наук: 07.00.09`.
- **Место защиты:** Если заполнено поле **«Университет»** (`university`), оно добавляется в сведения об ответственности.

### 5. Веб-страница (`webpage`)
- Учитываются: Автор, Название страницы, Название сайта, URL, Дата доступа.

### 6. Рукопись (`manuscript`)
> Данный тип используется **только для подстрочных ссылок** на архивные документы. Записи типа `manuscript` **не включаются** в итоговый список литературы (Bibliography).

#### Правила отображения архивных ссылок:

- **Первичная ссылка (`first`):**
    Выводятся: **Полное название архива** (поле `publisher`), **Сокращение архива** в скобках (поле `archive`), **Поисковые данные** (поле `archive_location`).
- **Повторная ссылка (`subsequent`):**
    Выводятся: **Сокращение архива** (поле `archive`) и **Поисковые данные** (поле `archive_location`).

#### Различие «Дело» и «Документ»:
Поведение зависит от заполнения поля **«Тип рукописи»** (`genre`):

- **Если `genre = Дело`:**
    Если не указан локатор (номер листа), выводится общее количество листов из поля `number-of-pages`.
    *Пример вывода:* `23 л.`
- **Если `genre` пустое или `Документ`:**
    Если не указан локатор, выводится диапазон листов из поля `number-of-pages`.
    *Пример вывода:* `Л. 23.` `Л. 14–15.`
