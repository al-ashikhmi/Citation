Стиль основан на ГОСТ Р 7.0.5-2008 «Библиографическая ссылка. Общие требования и правила составления» и разработан для публикаций в научном журнале «Loci Imperiales», издаваемом в ГМЗ «Петергоф». Стиль позволяет корректно цитировать литературу и архивные источники в подстрочных сносках (Notes), а также составлять список использованной литературы (Bibliography).

Сортировка в списке литературы производится по алфавитному порядку. Сначала идут издания на русском языке, затем — на иностранных. Второй уровень — по автору/заглавию. Третий уровень — по дате издания.

Добавлена локализация стандартных обозначений на русский язык с некоторыми особенностями:
  - Составитель выводится при заполненном поле «Редактор серии» (collection-editor);
  - Публикатор выводится при заполненном поле «Контрибьютор» (contributor).
Локализация английских терминов не реализована на данный момент.

Добавлены элементы форматирования:
- Курсив для имен авторов (italic)
- Неразрывные пробелы (&#160;):
  - у авторов между инициалами, а также между фамилией и инициалами
  - перед разделителями — косой чертой (/) и двойной косой черттой (//)
  - в численных данных (Т.&#160;, №&#160;?? С.&#160;, &#160;с.)
  - в повторных ссылках (Там&#160;же.)
Комплексные ссылки разделяются точкой с запятой (;). На данный момент ссылки на одного автора не группируются.

## Положение ссылки
Стиль различает положение ссылки (цитирования):
  - первичная (first) — выводит полные данные, кроме «Издателя» (publisher) для книги, главы, статьи в сборнике конференции;
  - повторная (subsequent) — если заполнено поле «Краткое назв.», выводится оно. Если не заполнено, выводится полное название;
  - повторная, идущая сразу после первой (ibid) — выводит «Там же»;
  - повторная, идущая сразу после первой, но с другим локатором (ibid-with-locator)
К сожалению, нельзя прописать градацию для повторных ссылок. Если в новой ссылке отличается какой-то один элемент, кроме страниц, то ссылка не будет считаться ibid. Она будет или first, или subsequent.

## Типы записей

Стиль учитывает следующие типы записей в Zotero:

1) Книга (book)
 Учитываются авторы, редакторы, составители, переводчики, публикаторы.
 Для многотомных изданий указываются (при наличии) общее количество томов и данные конкретного тома.
 Для сериальных изданий указывается название серии и номер выпуска.

2) Глава или раздел книги (chapter) и статья в сборнике конференции (paper-conference)
 Если при цитировании не указан локатор, то выводится диапазон страниц раздела/статьи.

3) Статья в журнале (article)
 Учитывается автор, название статьи, название журнала, том, номер, год издания, страницы. Если не указан локатор, то выводится диапазон всей статьи.

4) Диссертация и автореферат (thesis)
 Обозначение ресурса (например, «автореф. дис. ... канд. ист. наук: 07.00.09») заполняется в поле «Тип» (genre).
 Заполненное поле «Университет» позволит добавить место защиты в сведения об ответственности.

5) Веб-страница (webpage)
 Учитываются автор, название страницы, название сайта, URL-адрес, дата доступа.

6) Рукопись (manuscript)
 Данный тип записи используется для создания ссылок (цитат) на архивные документы и дела. Он не учитывается в списке литературы (библиографии).
 
Ссылка на архивный источник различает положение ссылки:
- первичная (first). Выводятся полное название архива, его сокращение в скобках, поисковые данные. Для этого необходимо заполнять поля: «Учреждение» (publisher, полное название архива), «Архив» (archive, сокращение), «Место в архиве» (archive_location, поисковые данные).
- повторная (subsequent). Выводятся сокращение архива и поисковые данные.

Ссылка различает документы и дела:
- дело (поле «Тип рукописи» (genre) заполнено как «Дело»). В этом случае, если не указан локатор, будет выведено количество листов (number-of-pages), например: 23 л.
- документ (поле «Тип рукописи» (genre) пустое или заполнено как «Документ»). В этом случае, если не указан локатор, будет выведен диапазон листов из того же поля (number-of-pages), например: Л. 14–15.
